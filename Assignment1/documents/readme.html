Usage

$ant -p to list all targets specified by build.xml

Tasks

There are only two tasks: Mandelbrot Set and Traveling Salesman Problem

The default value of each can be found in the client files

Build

$ant compile to compile the program

$ant jar to generate the executable far file

$ant javadoc to generate the javadoc

$ant clean to clean the .class files

Run

$ant runServer to run the server

$ant runClientEuclideanTsp to run the TSP task

$ant runClientMandelbrotSet to run the Mendelbrot Set task
