Usage

To list all targets specified by build.xml:

$ant -p

Tasks

There are only two tasks: Mandelbrot Set and Traveling Salesman Problem

The default value of each can be found in the client files

Build

To compile the program:

$ant compile

To generate the executable jar files:

$ant jar

To generate the javadoc:

$ant javadoc

To clean the files:

$ant clean

Run

To run the server:

$ant runServer

To run the TSP task:

$ant runClientEuclideanTsp

To run the Mendelbrot Set task:

$ant runClientMandelbrotSet

To execute the jar files manually:

First:
Copy the dist/ directory

To run the server:

$java -Djava.rmi.server.useCodebaseOnly=false -Djava.security.policy=policy -cp dist/computer.jar computer.ComputerImpl

To run the mandelbrot set task

$java -Djava.rmi.server.codebase=http://www.cs.ucsb.edu/~yanxi/CS190B/ -Djava.security.policy=policy -cp dist/clients.jar clients.ClientMandelbrotSet

To run the traveling salesman problem task

$java -Djava.rmi.server.codebase=http://www.cs.ucsb.edu/~yanxi/CS190B/ -Djava.security.policy=policy -cp dist/clients.jar clients.ClientEuclideanTsp

Troubleshooting

If you see an error saying port already in use, try to change another port by editting the PORT variable in api.Computer class
